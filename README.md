# About
* Notes application is a note taking application created using Reactjs

# Technologies used
* Reactjs
* CSS

# Features
* Create the notes with title
* Read the notes
* Save the notes
* Pin the notes
* Save pin using color 
* Filter the note pins using categories
* Delete the pin
